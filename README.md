## Características Principales

* **Catálogo Dinámico:** Los productos (libros) se cargan de forma asíncrona desde un archivo `products.json`, haciendo peticiones http.
* **Carrito de Compras:** Funcionalidad completa para agregar, eliminar y ver productos en el carrito.
* **Diseño Responsivo:** La interfaz se adapta a diferentes tamaños de pantalla, desde móviles hasta escritorio.

---

## Tecnologías Utilizadas

Este proyecto fue construido utilizando las siguientes tecnologías:

* **HTML5:** Para la estructura y el marcado semántico del contenido.
* **CSS3:** Para el diseño, la maquetación (layout) y el estilo visual.
* **JavaScript:** Para la interactividad, la lógica del carrito y la carga de datos.
